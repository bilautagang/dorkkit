# dorkkit

How It Works

Enter a target domain (e.g. example.com)
Click Generate — all 90+ dork queries are instantly populated with your domain
Click ↗ Search on any dork to run it live on Google
Use filters and search to narrow down what you need

Key Features

Direct Google links — every dork opens a real Google search in a new tab
Open All in Tabs — run every visible dork at once
Tag filtering — filter by Recon, Files, Login, Vuln, or OSINT
Keyword search — find specific dorks instantly
Copy button — copy any query to clipboard
Keyboard shortcuts — G to focus domain, / to search, A to expand all
Ethical use disclaimer — built-in reminder for responsible, authorized use only
